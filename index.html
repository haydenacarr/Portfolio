Projects

GALACTIC SUPERCOMPUTING: An N-Body Simulation

Description: An N-body simulation using high-performance computing and real-time 3D visualization. This project models gravitational interactions between bodies using the Barnes-Hut algorithm and Leapfrog integration, accelerated with OpenMP, and visualized using OpenGL. Developed as a third-year group project.

Reflection: I am particularly proud of this simulation we made, as it required extensive study of mathematical papers and collaborative problem solving to accurately implement the algorithms.

This was my first experience working with a graphics API, which was more of a learning exercise, the primary achievement was in translating complex mathematical models into efficient, functional code.

Languages: C, C++

APIs: OpenGL, OpenMP
