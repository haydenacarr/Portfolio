GALACTIC SUPERCOMPUTING: An N-Body Simulation

Description: An N-body simulation using high-performance computing and real-time 3D visualisation. This project models gravitational interactions between bodies using the Barnes-Hut algorithm and Leapfrog integration, accelerated with OpenMP, and visualised using OpenGL. Made as a third year group project.

My thoughts: This project was my first time using a graphics API so that part is not impressive, but I am proud of the simulation itself. As it took alot of reading maths papers and group discussions to turn it into code.

APIs: OpenGL, OpenMP

Languages: C, C++
